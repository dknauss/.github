# Code of Conduct

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful in technical disagreement
- Assuming good intent while still being precise about problems
- Giving actionable feedback
- Respecting boundaries, time, and expertise

Examples of unacceptable behavior include:

- Harassment, threats, or personal attacks
- Discriminatory or demeaning language
- Publishing private information without permission
- Repeatedly derailing technical discussions

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing the project or maintainer.

## Enforcement

Report unacceptable behavior to the maintainer through the contact methods listed on [Dan Knauss's profile](https://github.com/dknauss) or [dan.knauss.ca](https://dan.knauss.ca).

The maintainer may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
